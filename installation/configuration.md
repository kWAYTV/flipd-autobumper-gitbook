# Configuration file

```yaml
# Tool settings
bumpEach: 1800 # Time in seconds between bumps (minimum 1800 seconds or 30 minutes)
headless:  # Toggle for showing (False) or hiding (True) the browser window
nickname: "" # Any username
quote_switch:  # Toggle for sending The Simpsons quotes with bump messages
chrome_version:  # Your local Chrome version
user_agent: "" # User agent string, found on whatismybrowser.com

# Discord settings
webhook_switch:  # Toggle for sending Discord webhooks
webhook_url: "" # Webhook URL (used if webhooks are enabled)
```
